# АБ тест
В данном репозитории представлен анализ учебных данных по проведению АБ теста, взятый с https://www.kaggle.com/datasets/sergylog/ab-test-aggregated-data/data

Целью анализа была оценка эффективности двух групп по ключевым метрикам.

В работе был проанализирован вид распределения, поведен непараметрический тест и оценка устойчивости через Bootstrap с построением доверительных интервалов.
